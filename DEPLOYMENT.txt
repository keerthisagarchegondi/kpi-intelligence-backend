# Production Deployment Guide

## Quick Start

### Using PowerShell (Windows)

```powershell
# Deploy production environment
.\manage.ps1 production-deploy

# Run integration tests
.\manage.ps1 test-local

# Check health status
.\manage.ps1 health-check
```

### Using Make (Linux/Mac)

```bash
# Deploy production environment
make production-deploy

# Run integration tests
make test-local

# Check health status
make health-check
```

## Docker Commands

### Start Services
```bash
docker-compose up -d
```

### Stop Services
```bash
docker-compose down
```

### View Logs
```bash
docker-compose logs -f
```

### Run Tests in Docker
```bash
docker-compose --profile test up test
```

## Environment Configuration

The `.env` file contains all production settings:
- Database credentials
- API configuration
- CORS settings
- Logging levels

## Health Checks

API Health: http://localhost:8000/health
API Docs: http://localhost:8000/docs
